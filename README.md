# SiteCode React Starter

A new React + Vite website starter built in a proper front-end stack structure.

## Project setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
4. Preview the production build:
   ```bash
   npm run preview
   ```

## What is included

- React 18
- Vite development tooling
- Component-based layout in `src/components`
- Global base styles in `src/index.css`
- Simple homepage structure with header, hero, and footer
